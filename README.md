# Creep

Automatically scrapes a website, and returns the sitemap as a hash.

## Installation

```sh
$ gem install 'load'
$ load christianbundy/creep
$ cd creep
$ bundle install
```

## Usage

```sh
$ bin/creep # creeps on https://www.joingrouper.com/
$ bin/creep 'http://example.com'
```

## Documentation

```sh
$ yard server
```

## Testing

```sh
$ rake
```
